## Getting Started

1. clone the project

   ```terminal
   git clone git@github.com:samirkhanal077/sajilo-yatayat-flutter.git
   ```

2. download dependiencies

   ```terminal
   flutter pub get
   ```

3. Run The project

   ```terminal
   flutter run
   ```
